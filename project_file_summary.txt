สรุปหน้าที่ของแต่ละไฟล์ในโปรเจกต์ TN Messenger Service:

1. 404.html
   - หน้าแสดงเมื่อไม่พบหน้าที่ร้องขอ (Page Not Found)

2. account_list.html
   - แสดงข้อมูลบัญชีผู้ใช้ที่ล็อกอินอยู่ พร้อมรายละเอียดบัญชี

3. customer_dashboard.html
   - Dashboard สำหรับลูกค้า แสดงปุ่มนำทางไปยังฟีเจอร์ต่าง ๆ เช่น สร้างใบสั่งงาน, เช็คสถานะ, ข่าวสาร และแสดงรายชื่อสาขา

4. customer_workorder.html
   - ฟอร์มสำหรับลูกค้าเพื่อกรอกข้อมูลและสร้างใบสั่งงานใหม่

5. dashboard.html
   - หน้า Login และ Dashboard หลักของระบบ (หลังล็อกอินจะแสดงเมนูหลักสำหรับผู้ใช้)

6. index.html
   - ฟอร์มสำหรับสร้างใบสั่งงาน (คล้าย customer_workorder.html แต่ลิงก์กลับ Dashboard หลัก)

7. news_page.html
   - แสดงข่าวสารจาก Google Sheet ในรูปแบบรายการข่าว

8. register.html
   - ฟอร์มลงทะเบียนผู้ใช้ใหม่ (สมัครสมาชิก)

9. sdknfklansdlfnaosdbgoiadotablelist.html
   - (ชื่อไฟล์สุ่ม) ใช้สำหรับแสดงข้อมูลตารางจาก Google Sheet (รายละเอียดเฉพาะทาง)

10. tracking.html
    - หน้าให้ลูกค้าค้นหาและติดตามสถานะใบสั่งงานด้วยเลขที่ใบสั่งงาน

11. tracking_2.html
    - หน้าแสดงสถานะงาน (เวอร์ชัน 2) สำหรับติดตามงานด้วยเลขที่ใบสั่งงาน

12. tracking_admin.html
    - หน้าแอดมินสำหรับค้นหาและติดตามสถานะใบสั่งงาน พร้อมปุ่มกลับ Dashboard

13. update-status.html
    - ฟอร์มสำหรับอัปเดตสถานะใบสั่งงาน เช่น เปลี่ยนสถานะ/เพิ่มหมายเหตุ โดยกรอกเลขที่ใบสั่งงาน

14. font/
    - โฟลเดอร์เก็บไฟล์ฟอนต์ที่ใช้ในโปรเจกต์
